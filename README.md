## Balance Mod

This mod adjusts balance from 2.0

- Koji drone damage taken (`damageMultiplier`) 50% -> 85%
- Giant
    - movement 5 -> 3
    - attack cleaves in shape like this
        ```
        - - X
        - G X
        - - X
        ```
